# tinker-dpaint

DPaint plugin for [TINKER](https://github.com/liriliri/tinker).

## About

This plugin integrates [DPaint](https://github.com/steffest/DPaint-js), a recreation of the classic Deluxe Paint program, into TINKER. DPaint is a legendary bitmap graphics editor originally created by Electronic Arts for the Amiga computer, now reimagined for modern web browsers.

## Installation

```bash
npm install -g tinker-dpaint
```
