# tinker-ctool

Ctool plugin for [Tinker](https://github.com/liriliri/tinker).

## About

This plugin integrates [Ctool](https://github.com/baiy/Ctool), a comprehensive toolkit for developers, into Tinker. Ctool provides a wide range of utilities for common development tasks including encoding/decoding, encryption/decryption, format conversion, and many other developer tools.

## Installation

```bash
npm install -g tinker-ctool
```
