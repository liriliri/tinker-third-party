# tinker-svgedit

SVGEdit plugin for [Tinker](https://github.com/liriliri/tinker).

## About

This plugin integrates [SVG-Edit](https://github.com/SVG-Edit/svgedit), a powerful web-based SVG editor, into Tinker. SVG-Edit provides a complete vector graphics editing experience with support for drawing shapes, paths, text, and advanced SVG features like gradients, patterns, and filters.

## Installation

```bash
npm install -g tinker-svgedit
```
