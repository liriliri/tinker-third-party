# tinker-hoppscotch

Hoppscotch plugin for [Tinker](https://github.com/liriliri/tinker).

## About

This plugin integrates [Hoppscotch](https://github.com/hoppscotch/hoppscotch), an open source API development ecosystem, into Tinker. Hoppscotch provides a powerful interface for testing and debugging REST, GraphQL, WebSocket, and other API protocols with features like request collections, environment variables, authentication helpers, and real-time response visualization.

## Installation

```bash
npm install -g tinker-hoppscotch
```
